# Воркшоп: Разработка логики игры "Виселица"

Воркшоп по модулю "Основы программирования" курса "Фронтенд-разработчик" в онлайн-школе Нетология. 
Реализация логики игры "Виселица" на JavaScript.

📌 Посмотреть проект онлайн на [GitHub Pages](https://potykalov.github.io/hangmanGame/).

 ## 📝 Описание

 🎯 **Суть игры**

Слово выбирается случайно из массива.<br>
Игрок пытается угадать слово, нажимая буквы. 

🕹️ **Как играть**

- Слово выбирается случайно из массива и обозначается его длина с помощью звездочек: **`* * * * *`**.
- Игрок начинает нажимать буквы (только РУССКАЯ раскладка).
- Если буква есть в слове — она обводится и открывается на своих местах: **`* A * * A`**.
- Если буквы нет — она зачеркивается и засчитывается ошибка.

☠️ **Правило "Виселицы"**

За каждую ошибку рисуется часть человечка (голова, тело, руки, ноги).<br>
Дается 7 попыток.
  
🏁 **Конец игры**

**Победа:** слово полностью угадано.<br>
**Поражение:** человечек нарисован полностью раньше, чем слово раскрыто.

## Существующие функции

Отрисовывает состояние игрового персонажа. Принимает количество ошибок.
```js
drawPerson(number)
```

Отрисовывает состояние клавиатуры, которое было передано аргументом.
```js
drawBoard([{char: string, success: boolean, error: boolean}])
```

Отрисовывает состояние ответа.
```js
drawAnswerState(string)
```

Отображает победу игрока, после чего запустит новую игру.
```js
winGame()
```

Возвращает начальное состояние клавиатуры.
```js
getDefaultKeyboard()
```

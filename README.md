# 🎮 Игра "Виселица"

Проект выполнен в рамках воркшопа по модулю **"Основы программирования"** курса *Frontend-разработчик* от Нетологии.

📌 **Демо проекта:** [GitHub Pages](https://potykalov.github.io/hangmanGame/)  

---

## 📝 Описание

Реализация классической игры **"Виселица"** на JavaScript.

Игроку необходимо угадать слово, выбирая буквы.  
За каждую ошибку рисуется часть виселицы.

---

## 🎯 Функциональность

- Случайный выбор слова из словаря
- Отображение текущего состояния слова (`* * *`)
- Обработка нажатий на клавиатуру
- Подсветка:
  - ✅ правильных букв
  - ❌ неправильных букв
- Подсчет ошибок
- Отрисовка человечка по мере ошибок
- Определение:
  - 🏆 победы
  - 💀 поражения
- Автоматический рестарт игры

---

## 🕹️ Как играть

- Слово скрыто и отображается как: `* * * *`
- Игрок выбирает буквы:
  - если буква есть — она открывается
  - если нет — увеличивается счетчик ошибок
- Максимум: **7 ошибок**

## ⚙️ Реализованные функции

### Работа с игрой

```js
generateWord()   // генерация случайного слова
startGame()      // запуск новой игры
onKeyClick()     // обработка клика по букве
```

### Отрисовка (предоставлены)

```js
drawPerson(number)        // Отрисовывает состояние игрового персонажа. Принимает количество ошибок
drawBoard(state)          // Отрисовывает состояние клавиатуры, которое было передано аргументом
drawAnswerState(string)   // Отрисовывает состояние ответа
winGame()                 // Отображает победу игрока, после чего запустит новую игру
getDefaultKeyboard()      // Возвращает начальное состояние клавиатуры
```
